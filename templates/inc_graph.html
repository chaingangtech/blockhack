The dropdowns allow you to select a base currency and a counter currency. Select the currency first, and the issuer dropdown will update to show which issuers are available for the selected currency. On the right side, the current ledger number is represented, followed by a circle which indicates whether or not Ripple Charts is currently connected to the Ripple Network real time feed.

This chart below shows price change over time. Grey bars in the background represent trade volume for that time period. In the top left, select time intervals between each data point. The displayed time range will automatically adjust based on which interval you select. At the top right you can select between line or candlestick display.

Hover your mouse over the chart to see open, high, low, and close values for each of the data points. To download a csv file of the data from the current chart, click the icon to the right of the currency dropdowns.

Order Book

The current order book for the selected currency pair is below - these are orders across the Ripple Network that have been placed but not yet filled. "Bids" are orders to buy the base currency, and "Asks" are orders to sell the base currency. The “Total” column shows the total amount of currency anyone can get at that price. The "Size" columns show the amount of base currency available to buy or sell at "Bid Price" or "Ask Price." The blue line shows the depth of the order book - the amount of base currency available at different rates. The vertical line represents the midpoint between the best bid and ask price which is the current market price.
